# どんなアプリか？
Slackのリマインドを作成するアプリです。ボタンをクリックするだけで簡単にリマインドのコマンドを作成することができます。

# なぜ作ったのか？
エンジニアに需要があるアプリを作成したいと考えたためです。Slackは必須ツールですが、リマインドの設定が難しいと感じていました。そのため、簡単にリマインドを設定できるツールがあれば便利だと考えてこのアプリを作成しました。

# 使用技術
HTML/CSS/Vue.js/Netlify
